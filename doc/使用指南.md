# SyncMark 使用指南

## 一、安装 Chrome 插件

### 1.1 安装步骤

1. **打开扩展程序管理页面**
   - 方式一：在地址栏输入 `chrome://extensions/`
   - 方式二：点击右上角菜单 → 更多工具 → 扩展程序

2. **开启开发者模式**
   
   在页面右上角开启"开发者模式"开关
   
   ![开启开发者模式](image.png)

3. **加载插件**
   
   点击"加载已解压的扩展程序"按钮
   
   ![加载扩展程序](image-1.png)

4. **选择插件文件夹**
   
   选择 SyncMark 插件文件夹，点击"选择文件夹"
   
   ![选择文件夹](image-2.png)

5. **固定插件图标**
   
   安装成功后，插件图标会出现在浏览器工具栏。点击固定图标，将其显示在工具栏，方便快速访问。
   
   ![固定图标](image-3.png)
   
   ![工具栏显示](image-4.png)

6. **打开插件**
   
   点击插件图标，打开 SyncMark 弹窗
   
   ![插件弹窗](image-5.png)

## 二、配置同步

### 2.1 选择同步平台

SyncMark 支持两个代码托管平台进行书签同步：

- **GitHub** - 全球最大的代码托管平台
- **Gitee (码云)** - 国内访问速度更快的代码托管平台

根据您的网络环境和使用习惯选择合适的平台。

### 2.2 配置 GitHub Token

1. **打开设置页面**
   
   点击插件头像按钮或底部设置按钮，打开配置界面
   
   ![打开设置](image-6.png)

2. **生成 GitHub Token**
   
   点击 GitHub Token 输入框旁的链接按钮，跳转到 GitHub Token 生成页面
   
   ![GitHub 配置](image-7.png)
   
   > 注意：如果未注册或登录 GitHub，请先完成注册和登录

3. **复制 Token**
   
   在 GitHub 页面点击生成按钮，复制生成的 Token
   
   ![生成 Token](image-8.png)
   
   > ⚠️ 重要：Token 只会显示一次，请妥善保存

4. **保存配置**
   
   将 Token 粘贴到输入框，点击"保存设置"
   
   ![保存设置](image-9.png)

### 2.3 配置 Gitee Token

1. **打开设置页面**
   
   点击插件头像按钮或底部设置按钮，打开配置界面

2. **生成 Gitee Token**
   
   点击 Gitee Token 输入框旁的链接按钮，跳转到 Gitee Token 生成页面
   
   ![Gitee 配置](image-16.png)
   
   ![生成 Token](image-17.png)
   
   > 注意：如果未注册或登录 Gitee，请先完成注册和登录

3. **保存配置**
   
   将 Token 粘贴到输入框，点击"保存设置"
   
   ![保存设置](image-19.png)
   
   ![配置完成](image-18.png)

### 2.4 Token 验证

- 如果 Token 填写错误，系统会提示验证失败
- 删除输入框中的 Token 可以清空配置
- 验证失败示例：

![验证失败](image-20.png)

## 三、使用同步功能

### 3.1 首次同步

配置完成后，首次使用会在插件图标上显示红色角标，提示有未同步的书签

![红色角标](image-10.png)

点击"同步"按钮，将本地书签上传到远程平台，角标消失

![同步完成](image-11.png)

![同步成功](image-12.png)

### 3.2 新增书签同步

当您收藏新书签时，插件会自动检测并显示红色角标

![新书签提示](image-13.png)

点击同步按钮即可上传新书签

### 3.3 删除书签同步

删除书签后，本地与远程数据不一致，也会显示红色角标

![删除提示](image-14.png)

点击同步按钮更新远程数据

### 3.4 从远程同步到本地

点击"远程同步到本地"按钮，可以将远程最新数据完全覆盖到本地

![远程同步](image-15.png)

> ⚠️ 注意：此操作会覆盖本地所有书签数据，请谨慎使用

## 四、常见问题

### 4.1 红色角标什么时候出现？

只要本地书签与远程数据不一致，就会显示红色角标，包括：
- 新增书签
- 删除书签
- 修改书签
- 首次配置完成

### 4.2 Token 验证失败怎么办？

1. 检查 Token 是否完整复制
2. 确认 Token 权限是否正确
3. 检查网络连接是否正常
4. 尝试重新生成 Token

### 4.3 如何切换同步平台？

1. 打开设置页面
2. 清空当前平台的 Token
3. 配置新平台的 Token
4. 保存设置即可
